 Analisis de una UI para una Aplicacion Web
 Pagina web seleccionada: Mercado Libre
 Mercado Libre es un un sistema digital que permite comprar y vender productos en línea; está dirigido a un público amplio que incluye personas que buscan comprar artículos nuevos o usados y también vendedores independientes o empresas.
 Su contexto de uso principal es el acceso desde computadoras o teléfonos móviles para explorar productos, comparar precios, realizar compras, vender artículos y realizar pagos de forma rápida y segura.
 • Header: El encabezado de Mercado Libre presenta una estructura clara y funcional: incluye el logo, que actúa como botón para volver a la página principal, un botón para seleccionar o ingresar la ubicación del usuario y una barra de búsqueda central que facilita encontrar productos. Además, cuenta con opciones para registrarse y para iniciar sesión en una cuenta, un botón para acceder al carrito de compras donde se revisan los productos seleccionados, y también una opción para ver el historial de compras realizadas por el usuario.
 
 • Menu: incluye opciones como categorías de productos como electrónica, hogar o deportes, cupones para , ofertas variadas, supermercado boton que contiene ofertas de alimentos y productos para el hogar , seccion de moda , mercado play,seccion dedicada a peliculas y series ,Seccion para vender objetos y una seccion de preguntas frecuentes.
